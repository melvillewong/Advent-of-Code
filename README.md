# Advent of Code

### Template

`./template`

```bash
template
├── bin
│   └── part1.rs
└── input.txt
```

```bash
cp -r ./template/. ./2022/day-01/src/.
rm ./2022/day-01/src/main.rs
```

### Script

`./create_template.sh <year> <date>`

To create template in `./<year>/<date>`

Example:

```bash
./create_template.sh 2022 day-01
```
